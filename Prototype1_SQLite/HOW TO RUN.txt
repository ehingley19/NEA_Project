1. Run this command in the console. This command defines the starting file for the application.

> set FLASK_APP=app.py

2. After, run this command in the console to start the web application.

> flask run

3. Once finished, press Ctrl-C in the console to stop it.

======================================
NOTE: The above assumes the following:

i. pip install flask
ii. pip install flask-wtf
======================================
